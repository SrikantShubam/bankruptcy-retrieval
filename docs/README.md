# Worktree D Starter Pack

This folder is the minimal handoff package for starting Worktree D (agent-first pipeline).

## Contents
- `ARCHITECTURE.md`: target pipeline structure and node responsibilities
- `ARCHITECTURE_BUNDLE_V1.md`: coverage-first bundle retrieval architecture for v4 ingestion
- `TELEMETRY.md`: required metrics, logging contract, and report format
- `AGENT_BRIEF.md`: operating brief for the implementation agent (scope, access, constraints)
- `BASELINE_AND_CHECKLIST.md`: current baseline, known error clusters, env/run commands, and Day-1 execution checklist
- `GITHUB_REPO_REQUIREMENTS.md`: required GitHub/repo structure, branch/worktree conventions, CI/test gates, and PR rules for Worktree D

## Purpose
Use this folder as the single source of truth when bootstrapping Worktree D.
